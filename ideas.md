# PyhgoShift 홈페이지 디자인 아이디어

## 선택된 디자인 철학: **테크 미니멀리즘 + 프리미엄 모던**

파이고시프트는 SI 프로젝트 관리의 최신 AI 에이전트 솔루션을 제공하는 회사입니다. 복잡한 기술을 직관적으로 전달하면서도 프리미엄한 느낌을 주어야 합니다.

### 핵심 원칙 (Core Principles)

1. **기술적 신뢰성**: 깔끔한 라인, 명확한 계층 구조, 정밀한 타이포그래피로 기술 전문성 표현
2. **공간의 우아함**: 넉넉한 여백과 전략적 배치로 프리미엄 느낌 전달
3. **동적 상호작용**: 미묘한 애니메이션과 전환으로 AI의 지능성과 반응성 표현
4. **명확한 정보 계층**: PMBOK 단계별 AI 도입 과정을 시각적으로 명확하게 표현

### 색상 철학 (Color Philosophy)

- **주색상**: 진한 네이비 (#0F172A) - 신뢰, 기술, 안정성
- **강조색**: 생생한 사이안 (#06B6D4) - 혁신, AI, 에너지
- **배경**: 거의 흰색 (#FAFAFA) - 깔끔함, 프리미엄
- **텍스트**: 차콜 그레이 (#1F2937) - 가독성, 전문성
- **악센트**: 라임 그린 (#10B981) - 성공, 진행 상태

**감정적 의도**: 기술적 신뢰성과 현대적 혁신성의 조화

### 레이아웃 패러다임 (Layout Paradigm)

- **비대칭 그리드**: 좌측 콘텐츠 영역, 우측 시각화 영역
- **Z-패턴 흐름**: 상단 헤더 → 중앙 히어로 → 하단 서비스 섹션
- **카드 기반 구조**: 각 PMBOK 단계를 독립적인 카드로 표현
- **플로팅 요소**: 배경에 추상적인 기하학 형태로 깊이감 표현

### 시그니처 요소 (Signature Elements)

1. **AI 흐름 다이어그램**: 프로젝트 단계별 AI 에이전트의 역할을 연결 선으로 표현
2. **진행 상태 인디케이터**: 원형 진행률 표시기로 각 단계의 완성도 표현
3. **기하학적 배경 패턴**: 미묘한 그리드와 대각선으로 기술적 느낌 강화

### 상호작용 철학 (Interaction Philosophy)

- **호버 효과**: 카드가 살짝 들어올려지고 그림자가 깊어짐
- **스크롤 애니메이션**: 섹션이 부드럽게 페이드인, 요소들이 순차적으로 나타남
- **CTA 버튼**: 클릭 시 물결 효과, 마우스 오버 시 색상 변화
- **탭 전환**: 매끄러운 슬라이드 전환으로 서비스 카테고리 변경

### 애니메이션 가이드라인

- **지속 시간**: 300-500ms (빠르고 반응적)
- **이징**: cubic-bezier(0.4, 0, 0.2, 1) (자연스러운 감속)
- **입장 애니메이션**: 아래에서 위로 페이드인 + 스케일 (0.95 → 1)
- **호버 상태**: 배경색 변화 + 약간의 상승 + 그림자 강화
- **로딩 상태**: 부드러운 회전 애니메이션 (무한 반복)

### 타이포그래피 시스템 (Typography System)

**폰트 조합**:
- **제목**: Poppins (Bold 700, 600) - 현대적이고 기술적
- **본문**: Inter (Regular 400, Medium 500) - 명확하고 가독성 높음
- **강조**: Poppins (SemiBold 600) - 중요 정보 강조

**계층 구조**:
- H1: Poppins 700, 48px (페이지 제목)
- H2: Poppins 600, 36px (섹션 제목)
- H3: Poppins 600, 24px (카드 제목)
- Body: Inter 400, 16px (본문)
- Small: Inter 400, 14px (보조 텍스트)

### 시각적 특징

- **그림자**: 미묘한 다층 그림자 (0 1px 3px, 0 4px 6px)
- **모서리**: 8px 라운드 (카드), 4px 라운드 (버튼)
- **테두리**: 1px 라이트 그레이 (#E5E7EB)
- **배경 패턴**: 미묘한 도트 또는 그리드 패턴 (opacity 5%)

## 페이지 구조

### 1. 헤더 (Navigation)
- 로고 (좌측)
- 네비게이션 메뉴 (중앙)
- CTA 버튼 (우측)

### 2. 히어로 섹션
- 큰 제목: "AI-Driven Project Management Excellence"
- 부제목: PMBOK 단계별 AI 에이전트 도입
- 배경: 추상적인 기하학 패턴 + 그라데이션
- CTA: "상담 예약" 버튼

### 3. 가치 제안 섹션
- 3개의 핵심 가치: 자동화, 지능화, 효율화
- 각각 아이콘 + 설명

### 4. PMBOK 단계별 AI 도입 (메인 섹션)
- 5개 단계 (기획, 실행, 모니터링, 종료, 통합)
- 각 단계별 카드: 아이콘, 제목, 설명, 기능 리스트
- 연결 선으로 흐름 표현

### 5. 서비스 카테고리
- 컨설팅 전략
- 맞춤형 수행
- 기술 지원

### 6. 사례 연구 / 성과
- 프로젝트 성공 사례
- 수치 기반 성과 (ROI, 시간 단축 등)

### 7. 연락처 / CTA
- 상담 신청 폼
- 연락처 정보

1. **1차**: 헤더, 히어로, 가치 제안
2. **2차**: PMBOK 단계별 섹션 (핵심)
3. **3차**: 서비스 카테고리, 사례 연구
4. **4차**: 연락처, 상호작용 고도화
